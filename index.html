ABSTRACT

The "Wireless Control System" project aims to develop a system that allows for the wireless control of electrical devices using an ESP32 microcontroller and a relay module. The project involves the use of four relays to control four different electrical loads, such as bulbs, from a remote location via Wi-Fi. This system is designed to be user-friendly and cost-effective, providing an efficient solution for home automation. The project has significant potential applications in residential and industrial settings where remote control of electrical devices is necessary.

INTRODUCTION

The Wireless Control System is a modern approach to home automation that leverages the capabilities of the ESP32 microcontroller. This system allows users to control electrical devices remotely through a Wi-Fi connection. The project is designed to demonstrate the practical applications of wireless communication in controlling electrical loads efficiently and reliably. The main objective is to provide a user-friendly and scalable solution for home and industrial automation.

OBJECTIVE

The objective of this project is to design and implement a wireless control system that allows users to control multiple electrical devices remotely using an ESP32 microcontroller and a relay module. The specific goals include:

WORKING PRINCIPLE

The working principle of the Wireless Control System is based on the ability of the ESP32 microcontroller to connect to a Wi-Fi network and receive control signals from a remote device, such as a smartphone or a computer. The ESP32 then processes these signals and activates the appropriate relays to control the connected electrical loads. The system uses a simple web interface or a mobile application to send commands to the ESP32, which in turn controls the relays.

SOFTWARE IMPLEMENTATION

The software implementation involves programming the ESP32 microcontroller to connect to a Wi-Fi network and listen for control signals. The microcontroller is programmed using the Arduino IDE, and the code includes functions to handle Wi-Fi connectivity, web server functionality, and relay control. Below is a simplified version of the code used in this project:


#include 

const char* ssid = "your_SSID";
const char* password = "your_PASSWORD";

WiFiServer server(80);

const int relay1 = 26;
const int relay2 = 27;
const int relay3 = 14;
const int relay4 = 12;

void setup() {
    Serial.begin(115200);
    WiFi.begin(ssid, password);
    
    while (WiFi.status() != WL_CONNECTED) {
        delay(1000);
        Serial.println("Connecting to WiFi...");
    }
    
    Serial.println("Connected to WiFi");
    server.begin();
    
    pinMode(relay1, OUTPUT);
    pinMode(relay2, OUTPUT);
    pinMode(relay3, OUTPUT);
    pinMode(relay4, OUTPUT);
    
    digitalWrite(relay1, HIGH);
    digitalWrite(relay2, HIGH);
    digitalWrite(relay3, HIGH);
    digitalWrite(relay4, HIGH);
}

void loop() {
    WiFiClient client = server.available();
    
    if (client) {
        String request = client.readStringUntil('\r');
        client.flush();
        
        if (request.indexOf("/relay1/on") != -1) {
            digitalWrite(relay1, LOW);
        } else if (request.indexOf("/relay1/off") != -1) {
            digitalWrite(relay1, HIGH);
        } else if (request.indexOf("/relay2/on") != -1) {
            digitalWrite(relay2, LOW);
        } else if (request.indexOf("/relay2/off") != -1) {
            digitalWrite(relay2, HIGH);
        } else if (request.indexOf("/relay3/on") != -1) {
            digitalWrite(relay3, LOW);
        } else if (request.indexOf("/relay3/off") != -1) {
            digitalWrite(relay3, HIGH);
        } else if (request.indexOf("/relay4/on") != -1) {
            digitalWrite(relay4, LOW);
        } else if (request.indexOf("/relay4/off") != -1) {
            digitalWrite(relay4, HIGH);
        }
        
        client.println("HTTP/1.1 200 OK");
        client.println("Content-Type: text/html");
        client.println("");
        client.println("");
        client.println("");
        client.println("");
        client.println("");
        client.println("");
        client.println("");
        
        client.stop();
    }
}
